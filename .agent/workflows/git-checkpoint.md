---
description: Create a git checkpoint (commit)
---

1. Check the current status of the repository:
```bash
git status
```
2. Add all changes to the staging area:
// turbo
```bash
git add .
```
3. Create a commit with a message (provide a message when prompted or use a default):
// turbo
```bash
git commit -m "checkpoint: update project state"
```
4. Optionally push to the remote repository:
```bash
git push origin main
```
